第一章

微积分概述：历史与发展

17 世纪 -21 世纪 | 微积分的诞生与演变

什么是微积分

微积分（Calculus）是数学的一个基础分支，主要研究变化和累积的数学理论。 它由微分学和积分学两大部分组成，通过极限概念将两者统一起来。

微积分的核心思想是"以直代曲"和"无限细分"： 将复杂的曲线问题转化为简单的直线问题，将整体的累积问题转化为无穷小量的求和问题。

📐 微积分的两大基本问题
  • 微分问题：已知函数，求其变化率（导数）
  • 积分问题：已知变化率，求原函数或累积量
这两个问题通过微积分基本定理联系起来，成为微积分的核心。

第二章

极限理论：微积分的基石

极限概念是微积分的理论基础，所有微积分概念都建立在极限之上

极限的定义

极限是微积分的核心概念，描述了函数或数列在某种趋势下的行为。 柯西在 1821 年给出了极限的严格定义，即著名的ε-δ 语言。

limx→a f(x) = L ⟺ ∀ε > 0, ∃δ > 0, 当 0 < |x - a| < δ 时，|f(x) - L| < ε
函数极限的 ε-δ 定义
limn→∞ an = L ⟺ ∀ε > 0, ∃N ∈ ℕ, 当 n > N 时，|an - L| < ε
数列极限的定义

重要极限

极限公式 名称 应用场景
limx→0 (sin x)/x = 1 第一重要极限 三角函数极限计算
limx→∞ (1 + 1/x)x = e 第二重要极限 指数函数、复利计算
limx→0 (ex - 1)/x = 1 指数极限 指数函数求导
limx→0 (ln(1+x))/x = 1 对数极限 对数函数求导

连续性

连续性是极限概念的直接应用。函数在某点连续，意味着函数在该点的极限值等于函数值。

f(x) 在 x = a 处连续 ⟺ limx→a f(x) = f(a)
连续性的定义
✅ 连续函数的性质
  • 最值定理：闭区间上的连续函数必有最大值和最小值
  • 介值定理：连续函数在区间内取到端点值之间的所有值
  • 零点定理：若 f(a)·f(b) < 0，则存在 c∈(a,b) 使 f(c) = 0
  • 一致连续性：闭区间上的连续函数必一致连续

无穷小量与无穷大量

无穷小量是极限为 0 的变量，无穷大量是极限为∞的变量。 它们是微积分中描述变化趋势的重要工具。

概念 定义 记号
无穷小量 lim α(x) = 0 α(x) = o(1)
无穷大量 lim f(x) = ∞ f(x) → ∞
等价无穷小 lim α(x)/β(x) = 1 α(x) ~ β(x)
高阶无穷小 lim α(x)/β(x) = 0 α(x) = o(β(x))
第三章

微分学：变化率的数学

导数与微分是研究函数变化规律的核心工具

导数的定义

导数描述了函数在某点的瞬时变化率，是微分学的核心概念。 几何上，导数表示曲线在该点的切线斜率。

f'(x) = limΔx→0 [f(x + Δx) - f(x)] / Δx = dy/dx
导数的定义（牛顿 - 莱布尼茨记号）
df = f'(x)dx
微分的定义

基本求导公式

函数 f(x) 导数 f'(x) 函数 f(x) 导数 f'(x)
C (常数) 0 xn nxn-1
sin x cos x cos x -sin x
ex ex ax ax ln a
ln x 1/x loga x 1/(x ln a)
tan x sec² x arcsin x 1/√(1-x²)

求导法则

(u ± v)' = u' ± v'     (uv)' = u'v + uv'     (u/v)' = (u'v - uv')/v²
四则运算求导法则

链式法则（复合函数求导）
📐 高阶导数

二阶导数： f''(x) = d²y/dx²，描述变化率的变化率（加速度）
n 阶导数： f(n)(x) = dny/dxn
莱布尼茨公式： (uv)(n) = Σ C(n,k)·u(k)·v(n-k)

微分中值定理

微分中值定理是微分学的理论基础，建立了函数值与导数值之间的联系。

罗尔定理：f(a) = f(b) ⟹ ∃c∈(a,b), f'(c) = 0
罗尔定理
拉格朗日中值定理：f(b) - f(a) = f'(c)(b - a), c∈(a,b)
拉格朗日中值定理
柯西中值定理：[f(b)-f(a)]/[g(b)-g(a)] = f'(c)/g'(c), c∈(a,b)
柯西中值定理

泰勒公式

泰勒公式将复杂函数展开为多项式，是微积分中最重要的公式之一， 在数值计算、物理建模等领域有广泛应用。

f(x) = f(a) + f'(a)(x-a) + f''(a)(x-a)²/2! + ... + f(n)(a)(x-a)n/n! + Rn(x)
泰勒公式（带余项）
ex = 1 + x + x²/2! + x³/3! + ... = Σ xn/n!
ex 的麦克劳林展开
sin x = x - x³/3! + x⁵/5! - x⁷/7! + ...

第四章

积分学：累积的数学

积分是微分的逆运算，用于计算面积、体积、累积量等

不定积分

不定积分是求原函数的运算，是微分的逆运算。 若 F'(x) = f(x)，则 F(x) 是 f(x) 的一个原函数。

∫ f(x)dx = F(x) + C, 其中 F'(x) = f(x)
不定积分的定义
✅ 基本积分公式
  • ∫ xndx = xn+1/(n+1) + C (n≠-1)
  • ∫ (1/x)dx = ln|x| + C
  • ∫ exdx = ex + C
  • ∫ sin x dx = -cos x + C
  • ∫ cos x dx = sin x + C
  • ∫ (1/(1+x²))dx = arctan x + C

定积分

定积分表示函数在区间上的累积和，几何上表示曲线与 x 轴围成的面积。 黎曼在 1854 年给出了定积分的严格定义。

∫ab f(x)dx = limn→∞ Σi=1n f(ξi)Δxi
黎曼积分的定义
∫ab f(x)dx = F(b) - F(a)
牛顿 - 莱布尼茨公式（微积分基本定理）

积分方法

方法 公式 适用场景
换元积分法 ∫ f(g(x))g'(x)dx = ∫ f(u)du 复合函数积分
分部积分法 ∫ udv = uv - ∫ vdu 乘积函数积分
部分分式法 有理函数分解为简单分式 有理函数积分
三角代换 √(a²-x²) → x = a sin θ 含根式的积分

重积分

重积分是定积分在多维空间的推广，用于计算体积、质量、质心等物理量。

二重积分：∬D f(x,y)dA = ∫ab ∫cd f(x,y)dydx
二重积分（直角坐标）
三重积分：∭V f(x,y,z)dV
三重积分
📐 坐标变换

极坐标： x = r cos θ, y = r sin θ, dA = r dr dθ
柱坐标： x = r cos θ, y = r sin θ, z = z, dV = r dr dθ dz
球坐标： x = ρ sin φ cos θ, y = ρ sin φ sin θ, z = ρ cos φ, dV = ρ² sin φ dρ dφ dθ

曲线积分与曲面积分

曲线积分和曲面积分是向量分析的基础，在电磁学、流体力学中有重要应用。

第一类曲线积分：∫L f(x,y)ds
对弧长的曲线积分
第二类曲线积分：∫L Pdx + Qdy
对坐标的曲线积分
高斯公式：∭V (∂P/∂x + ∂Q/∂y + ∂R/∂z)dV = ∯S (Pdydz + Qdzdx + Rdxdy)
高斯公式（散度定理）
斯托克斯公式：∬S (∂R/∂y - ∂Q/∂z)dydz + ... = ∮L Pdx + Qdy + Rdz

第五章

级数理论：无穷的和

级数是将无穷多个数或函数相加的数学工具

数项级数

数项级数是无穷多个数相加的和。级数的收敛性是级数理论的核心问题。

Σn=1∞ an = a1 + a2 + a3 + ...
数项级数的定义
⚠️ 收敛判别法
  • 必要条件：若级数收敛，则 lim an = 0
  • 比较判别法：与已知收敛/发散级数比较
  • 比值判别法：lim |an+1/an| = ρ
  • 根值判别法：lim √[n]{|an|} = ρ
  • 积分判别法：与反常积分比较
  • 交错级数：莱布尼茨判别法

函数项级数

函数项级数是无穷多个函数相加的和。一致收敛性是函数项级数的关键概念。

Σn=1∞ un(x) = u1(x) + u2(x) + u3(x) + ...
函数项级数
📐 一致收敛的性质

若函数项级数一致收敛，则：
• 和函数连续（若各项连续）
• 可以逐项求积分
• 可以逐项求导（若导数级数一致收敛）

幂级数

幂级数是形如 Σ an(x-a)n 的函数项级数， 是泰勒公式的无穷级数形式。

Σn=0∞ an(x-a)n = a0 + a1(x-a) + a2(x-a)² + ...
幂级数的一般形式
收敛半径：R = 1/limn→∞ |an+1/an|
收敛半径公式

傅里叶级数

傅里叶级数将周期函数展开为正弦和余弦函数的级数， 在信号处理、热传导等领域有广泛应用。

f(x) = a0/2 + Σn=1∞ (ancos nx + bnsin nx)
傅里叶级数展开
an = (1/π)∫-ππ f(x)cos nx dx,   bn = (1/π)∫-ππ f(x)sin nx dx

第六章

应用领域：从物理到 AI

微积分是现代科学和工程的基础工具

物理学应用

微积分是经典力学、电磁学、量子力学等物理理论的语言。 牛顿正是为了研究物理问题而发明了微积分。

物理量 微积分表达 说明
速度 v = dx/dt 位移对时间的导数
加速度 a = dv/dt = d²x/dt² 速度对时间的导数
功 W = ∫ F·ds 力沿路径的积分
电场 E = -∇V 电势的负梯度
麦克斯韦方程 ∇·E = ρ/ε₀, ∇×E = -∂B/∂t 电磁学基本方程

工程学应用

  • 结构分析：计算梁的弯曲、应力分布
  • 流体力学：纳维 - 斯托克斯方程描述流体运动
  • 热传导：热传导方程描述温度分布
  • 控制系统：微分方程建模动态系统
  • 信号处理：傅里叶变换分析信号频谱

经济学应用

  • 边际分析：边际成本 = dC/dq，边际收益 = dR/dq
  • 最优化：利润最大化、成本最小化
  • 弹性分析：价格弹性 = (dQ/Q)/(dP/P)
  • 连续复利：A = Pert
  • 消费者剩余：CS = ∫0Q (D(q) - P)dq

机器学习与 AI

微积分是现代机器学习和人工智能的数学基础。 梯度下降、反向传播等核心算法都依赖于微积分。

梯度下降：θnew = θold - η·∇L(θ)
梯度下降算法（η 为学习率）
反向传播：∂L/∂w = ∂L/∂y · ∂y/∂w
链式法则在神经网络中的应用
✅ 微积分在 AI 中的核心应用
  • 优化算法：梯度下降、Adam、RMSprop 等
  • 反向传播：计算损失函数对权重的梯度
  • 卷积神经网络：卷积运算本质是积分
  • 概率模型：概率密度函数的积分
  • 变分推断：泛函极值问题

📜 总结：微积分的智慧传承

微积分是人类智慧的伟大结晶，从牛顿和莱布尼茨的开创性工作，到柯西、黎曼的严格化， 再到现代的泛函分析、微分几何，微积分经历了三百多年的发展， 已成为现代科学和工程不可或缺的基础工具。

微积分的核心思想：

  • 极限思想：用有限逼近无限，用直线逼近曲线
  • 变化与累积：微分描述变化，积分描述累积
  • 局部与整体：通过局部性质推导整体性质
  • 离散与连续：级数连接离散与连续
  • 抽象与应用：从具体问题抽象出一般理论，再应用于新问题

在 21 世纪的今天，微积分依然是物理学、工程学、经济学、计算机科学等领域的基础。 从经典力学到量子场论，从传统工程到人工智能，微积分的思想方法无处不在。 学习微积分，不仅是掌握数学工具，更是培养抽象思维、逻辑推理、问题解决能力的重要途径。
