1.2 批量重命名实战

让我们从最基础也是最实用的批量重命名开始。想象一下，你刚下载了 50 个课程视频，文件名都是乱码或者无意义的数字...

📋 实战场景：
你需要将下载文件夹中的所有视频文件按照"课程名称_章节号_标题.mp4"的格式重命名

工作流配置

# ~/.openclaw/workflows/file_rename.yml
name: batch_file_rename
description: 批量重命名下载文件夹中的视频文件
trigger: "manual"  # 手动触发，也可改为定时触发

steps:
  - action: list_files
    params:
      directory: "~/Downloads"
      pattern: "*.mp4"
      recursive: false
  
  - action: analyze_filenames
    model: "deepseek-chat"
    prompt: |
      分析以下文件名，提取课程名称、章节号和标题信息：
      {{file_list}}
      
      返回 JSON 格式：
      {
        "course_name": "课程名称",
        "chapter": "章节号",
        "title": "标题"
      }
  
  - action: rename_files
    params:
      naming_pattern: "{{course_name}}_{{chapter}}_{{title}}.mp4"
      dry_run: true  # 先预览，确认无误后再执行
  
  - action: confirm_and_execute
    params:
      require_confirmation: true
      backup_original: true  # 保留原文件备份

执行效果

原始文件名 重命名后 处理时间

DSC_9876.mp4 Python 入门_02_基础语法.mp4 0.5 秒

1.3 智能文件分类系统

文件多了怎么办？让 AI 帮你自动分类整理！

🗂️ 智能分类工作流程
1
扫描文件
遍历指定文件夹，收集所有文件的基本信息（文件名、大小、类型、创建时间）
2
AI 分析内容
读取文件内容（文本文件）或元数据（图片/视频），理解文件主题和用途
3
智能分类
根据预设分类规则（工作/学习/生活/项目等），将文件归类到对应文件夹
4
移动归档
自动移动文件到目标文件夹，生成整理报告

配置文件示例

# ~/.openclaw/workflows/smart_file_sorter.yml
name: smart_file_sorter
description: 智能文件分类整理系统
trigger: "cron: 0 20 * * *"  # 每天晚上 8 点自动执行

classification_rules:
  categories:
    - name: "工作文档"
      keywords: ["合同", "报告", "方案", "会议", "项目"]
      file_types: [".docx", ".xlsx", ".pptx", ".pdf"]
      target_folder: "~/Documents/Work"
    
    - name: "学习资料"
      keywords: ["教程", "笔记", "课程", "论文", "文献"]
      file_types: [".pdf", ".md", ".txt", ".epub"]
      target_folder: "~/Documents/Learning"
    
    - name: "图片素材"
      keywords: ["设计", "海报", "logo", "banner"]
      file_types: [".jpg", ".png", ".gif", ".psd", ".ai"]
      target_folder: "~/Pictures/Design"
    
    - name: "视频资源"
      keywords: ["教程", "课程", "讲座", "演示"]
      file_types: [".mp4", ".mov", ".avi", ".mkv"]
      target_folder: "~/Videos/Courses"

steps:
  - action: scan_directory
    params:
      source: "~/Downloads"
      include_subdirs: false
  
  - action: analyze_content
    model: "deepseek-chat"
    params:
      sample_size: 1000  # 读取前 1000 字符进行分析
  
  - action: classify_files
    params:
      rules: "{{classification_rules}}"
      confidence_threshold: 0.7  # 置信度阈值
  
  - action: move_files
    params:
      create_missing_dirs: true
      handle_conflicts: "rename"  # 重命名冲突文件
  
  - action: generate_report
    params:
      format: "markdown"
      send_to: "feishu"
⚠️ 注意事项：
1. 首次运行建议设置dry_run: true预览分类结果
2. 重要文件请先备份再执行自动整理
3. 可根据实际需求调整分类规则和关键词

1.4 PDF 文件自动化处理

PDF 是办公和学习中最常见的文件格式。OpenClaw 可以帮你自动完成 PDF 的各种处理任务。

常见 PDF 处理场景

场景 传统方式 OpenClaw 自动化 时间节省
PDF 转 Word 打开转换网站→上传→等待→下载（5 分钟/个） 自动批量转换（10 秒/个） 97%
提取文字 手动复制粘贴或 OCR 识别（10 分钟/页） 自动 OCR+ 提取（30 秒/页） 95%
合并拆分 使用专业软件操作（15 分钟/次） 一键自动处理（1 分钟/次） 93%
添加水印 逐个文件手动添加（8 分钟/个） 批量自动添加（20 秒/个） 96%

PDF 处理工作流

# ~/.openclaw/workflows/pdf_processor.yml
name: pdf_auto_processor
description: PDF 文件自动化处理工作流
trigger: "new_file:*.pdf"  # 监测到新 PDF 文件时自动触发

processing_tasks:
  - task: extract_text
    enabled: true
    output_format: "markdown"
    save_to: "~/Documents/PDF_Texts"
  
  - task: convert_to_word
    enabled: true
    preserve_formatting: true
    save_to: "~/Documents/Word_Docs"
  
  - task: generate_summary
    enabled: true
    model: "deepseek-chat"
    summary_length: "medium"  # short/medium/long
  
  - task: add_watermark
    enabled: false  # 按需开启
    watermark_text: "内部资料"
    position: "center"
    opacity: 0.3

steps:
  - action: wait_for_pdf
    params:
      watch_directory: "~/Downloads"
  
  - action: ocr_if_needed
    params:
      language: "chi_sim+eng"  # 中英文混合
  
  - action: execute_tasks
    params:
      tasks: "{{processing_tasks}}"
      parallel: true  # 并行处理提升速度
  
  - action: organize_output
    params:
      create_index: true
      index_format: "markdown"
  
  - action: notify_completion
    params:
      channel: "feishu"
      message: |
        ✅ PDF 处理完成！
        📄 源文件：{{source_file}}
        📝 提取文字：{{text_file}}
        📊 生成摘要：{{summary_file}}
