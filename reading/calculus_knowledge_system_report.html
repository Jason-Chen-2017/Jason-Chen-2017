1.1 微积分的定义

微积分（Calculus）是数学的一个基础分支，主要研究函数的微分和积分，以及相关的概念和应用。它是研究变化率和累积量的数学工具，为描述自然界中的连续变化现象提供了精确的语言。

微积分基本定理：∫ab f(x)dx = F(b) - F(a)，其中 F'(x) = f(x)

1.2 两大核心分支

📈

微分学

研究函数的变化率和局部性质，核心概念包括导数、微分、切线等。用于解决最优化、速度、加速度等问题。

📊

积分学

研究函数的累积效应和整体性质，核心概念包括定积分、不定积分、面积、体积等。用于解决求和、面积、体积等问题。

📜 2. 发展历史

2.1 关键人物

🧮

艾萨克·牛顿 (Isaac Newton)

1643 - 1727 年 | 英国

独立发明微积分（称为"流数术"），将微积分应用于物理学，建立了经典力学体系。主要著作《自然哲学的数学原理》奠定了微积分的物理基础。

📐

戈特弗里德·莱布尼茨 (Gottfried Leibniz)

1646 - 1716 年 | 德国

独立发明微积分，创造了现代微积分符号系统（∫、dx/dy 等）。他的符号系统更加清晰，被广泛采用至今。

🔬

奥古斯丁·路易·柯西 (Augustin-Louis Cauchy)

1789 - 1857 年 | 法国

建立了微积分的严格理论基础，引入了极限的ε-δ定义，使微积分从直观走向严谨。

📚

卡尔·魏尔斯特拉斯 (Karl Weierstrass)

1815 - 1897 年 | 德国

完善了微积分的严格化，建立了实数理论，为现代分析学奠定了基础。"现代分析学之父"。

2.2 发展时间线

公元前 3 世纪

阿基米德使用穷竭法计算面积和体积，是积分思想的萌芽

泰勒公式发表，为函数展开提供工具

19 世纪后期

黎曼、勒贝格发展积分理论，扩展积分的适用范围

20 世纪

微积分广泛应用于物理、工程、经济、生物等各个领域

🔍 3. 极限理论

极限是微积分的基础概念

3.1 极限的定义

limx→a f(x) = L

ε-δ定义：∀ε > 0, ∃δ > 0, 使得当 0 < |x - a| < δ 时，有 |f(x) - L| < ε

3.2 重要极限

第一个重要极限

limx→0 (sin x)/x = 1

第二个重要极限

limx→∞ (1 + 1/x)x = e

极限运算法则

lim(f ± g) = lim f ± lim g
lim(f · g) = lim f · lim g
lim(f/g) = lim f / lim g (lim g ≠ 0)

3.3 连续性

函数 f(x) 在点 a 连续：limx→a f(x) = f(a)

💡 核心要点

  • 连续函数在闭区间上必有最大值和最小值（最值定理）
  • 连续函数在闭区间上可取到最大值和最小值之间的任何值（介值定理）
  • 初等函数在其定义域内都是连续的

📈 4. 导数与微分

导数描述函数的瞬时变化率，是微分学的核心概念。

4.1 导数的定义

f'(x) = limΔx→0 [f(x + Δx) - f(x)] / Δx

或：f'(x) = dy/dx = limh→0 [f(x + h) - f(x)] / h

4.2 几何意义

🔷 切线斜率

导数 f'(x₀) 表示曲线 y = f(x) 在点 (x₀, f(x₀)) 处的切线斜率。

切线方程：y - f(x₀) = f'(x₀)(x - x₀)

4.3 基本求导公式

函数 导数 函数 导数
f(x) = c (常数) f'(x) = 0 f(x) = xⁿ f'(x) = nxⁿ⁻¹
f(x) = sin x f'(x) = cos x f(x) = cos x f'(x) = -sin x
f(x) = eˣ f'(x) = eˣ f(x) = ln x f'(x) = 1/x
f(x) = aˣ f'(x) = aˣ ln a f(x) = tan x f'(x) = sec²x

4.4 求导法则

四则运算法则

(u ± v)' = u' ± v'
(uv)' = u'v + uv'
(u/v)' = (u'v - uv')/v²

链式法则

dy/dx = dy/du · du/dx
[f(g(x))]' = f'(g(x)) · g'(x)

高阶导数

f''(x) = d²y/dx²
f⁽ⁿ⁾(x) = dⁿy/dxⁿ

4.5 导数的应用

🎯 主要应用领域

  • 最优化问题：求函数的最大值和最小值
  • 曲线分析：判断函数的单调性、凹凸性、拐点
  • 物理应用：速度、加速度、变化率分析
  • 经济学：边际分析、弹性分析
  • 近似计算：微分近似、泰勒展开

📊 5. 积分理论

积分是微分的逆运算，用于计算累积量和整体性质。

5.1 不定积分

∫ f(x)dx = F(x) + C，其中 F'(x) = f(x)

不定积分表示函数 f(x) 的全体原函数，C 为任意常数。

5.2 定积分

∫ab f(x)dx = limn→∞ Σi=1n f(ξᵢ)Δxᵢ

定积分表示函数在区间 [a,b] 上的累积和，几何意义是曲线下的面积。

5.3 微积分基本定理

🔑 牛顿 - 莱布尼茨公式

∫ab f(x)dx = F(b) - F(a)

该定理建立了微分与积分之间的联系，是微积分的核心定理。

5.4 积分方法

换元积分法

∫ f(g(x))g'(x)dx = ∫ f(u)du
其中 u = g(x)

分部积分法

∫ u dv = uv - ∫ v du

部分分式分解

将有理函数分解为简单分式
之和再逐项积分

5.5 积分的应用

应用领域 具体应用 公式/说明
几何 平面图形面积 A = ∫ab [f(x) - g(x)]dx
几何 旋转体体积 V = π∫ab [f(x)]²dx
几何 曲线弧长 L = ∫ab √(1 + [f'(x)]²)dx
物理 位移计算 s = ∫ v(t)dt
物理 功的计算 W = ∫ F·ds
经济 总成本/总收入 从边际函数求总量

∞ 6. 无穷级数

无穷级数是将无穷多个数相加的工具，是函数表示和近似计算的重要手段。

6.1 级数的定义

Σn=1∞ aₙ = a₁ + a₂ + a₃ + ...

部分和：Sₙ = Σk=1n aₖ
级数收敛：limn→∞ Sₙ = S（存在有限极限）

6.2 重要级数

几何级数

Σn=0∞ arⁿ = a/(1-r)
(|r| < 1 时收敛)

p-级数

Σn=1∞ 1/nᵖ
(p > 1 时收敛)

调和级数

Σn=1∞ 1/n
(发散)

6.3 泰勒级数

📐 函数的幂级数展开

f(x) = Σn=0∞ [f⁽ⁿ⁾(a)/n!] · (x-a)ⁿ

麦克劳林级数（a=0）：
f(x) = f(0) + f'(0)x + f''(0)x²/2! + f'''(0)x³/3! + ...

6.4 常见函数的泰勒展开

函数 泰勒展开式 收敛域
eˣ 1 + x + x²/2! + x³/3! + ... (-∞, +∞)
sin x x - x³/3! + x⁵/5! - x⁷/7! + ... (-∞, +∞)
cos x 1 - x²/2! + x⁴/4! - x⁶/6! + ... (-∞, +∞)
ln(1+x) x - x²/2 + x³/3 - x⁴/4 + ... (-1, 1]
1/(1-x) 1 + x + x² + x³ + ... (-1, 1)

🌐 7. 多元微积分

多元微积分研究多变量函数的微分和积分，是单变量微积分的推广。

7.1 偏导数

∂f/∂x = limΔx→0 [f(x+Δx, y) - f(x, y)] / Δx

∂f/∂y = limΔy→0 [f(x, y+Δy) - f(x, y)] / Δy

7.2 全微分

dz = (∂f/∂x)dx + (∂f/∂y)dy

7.3 方向导数与梯度

方向导数

∂f/∂l = ∇f · u
(u 为单位方向向量)

梯度

∇f = (∂f/∂x, ∂f/∂y)
指向函数增长最快的方向

7.4 多重积分

二重积分：∬D f(x,y)dA

三重积分：∭V f(x,y,z)dV

7.5 向量微积分

运算 定义 物理意义
梯度 (grad) ∇f = (∂f/∂x, ∂f/∂y, ∂f/∂z) 标量场的变化率
散度 (div) ∇·F = ∂Fₓ/∂x + ∂Fᵧ/∂y + ∂Fᵧ/∂z 向量场的源汇强度
旋度 (curl) ∇×F 向量场的旋转程度

7.6 重要定理

🔷 向量微积分基本定理

  • 格林公式：平面区域上的二重积分与边界曲线上的线积分的关系
  • 高斯公式（散度定理）：空间区域上的三重积分与边界曲面上的面积分的关系
  • 斯托克斯公式：曲面上的面积分与边界曲线上的线积分的关系
