1.1 认识你的"数字龙虾"

定义：OpenClaw 是一个开源的 AI 智能体（Agent）框架，它能够 24/7 不间断地在你的电脑上工作。与 ChatGPT 这类对话工具完全不同，OpenClaw 拥有"眼睛和手"：它可以控制浏览器、编写代码、阅读文件、执行命令，甚至在你睡觉时主动完成任务。

为什么叫"小龙虾"？

OpenClaw 的官方图标是一只红色的龙虾🦞，因此被社区亲切地称为"小龙虾"。这个昵称迅速传播开来，以至于部署 OpenClaw 被称为"养虾"，Skills 市场被称为"虾塘"，用户自称"虾农"。

🦞 龙虾文化：
• 养虾：部署和配置 OpenClaw 智能体
• 虾塘：ClawHub 技能市场
• 虾农：OpenClaw 用户和开发者
• 烹饪法：智能体调优方法
• 龙虾汤：智能体执行日志

核心特性：大脑 + 身体的完美结合

OpenClaw 的核心逻辑可以用一个简单的公式表示：

OpenClaw = 大脑（大模型） + 身体（执行框架）
  • 🧠 大脑：思考与决策

    需要接入 DeepSeek、通义千问、Claude、GPT 等大模型的 API 来负责思考和决策。大脑负责理解任务、制定计划、做出判断。

  • 🦾 身体：执行与操作

    OpenClaw 本身负责执行，通过安装各种"技能（Skill）"来获得不同的能力，就像一个工具箱。身体负责实际操作电脑、调用工具、完成任务。

1.2 与 ChatGPT 的本质区别

很多人第一次接触 OpenClaw 时会问："这和 ChatGPT 有什么区别？"这是一个非常好的问题。让我们通过对比来理解：

对比维度 ChatGPT / 传统大模型 OpenClaw 智能体
定位 对话式 AI 助手 自主执行型数字员工
交互模式 你问我答，被动响应 主动规划，持续执行
输出形式 文本、代码片段 实际完成的任务结果
执行能力 ❌ 无法直接操作电脑 ✅ 可控制浏览器、文件、API 等
记忆能力 单次会话内有效 持久化存储，跨会话记忆
工作时间 需要人工实时交互 7×24 小时自主工作
典型场景 写文案、查资料、学知识 自动化办公、数据处理、业务流程
"ChatGPT 像一个博学的顾问，能给你很好的建议，但不会帮你干活。OpenClaw 像一个得力的员工，你告诉它目标，它自己想办法完成。" —— OpenClaw 创始人 Peter Steinberger

实际案例对比

让我们通过一个具体场景来理解这种差异：

🎯 场景：每天早晨收集行业新闻并发送到飞书群

使用 ChatGPT 的方式

  1. 每天早上打开 ChatGPT 网页
  2. 输入："请帮我收集今天的人工智能行业新闻"
  3. 等待 ChatGPT 返回新闻列表（基于训练数据，可能不是最新）
  4. 手动复制新闻链接
  5. 打开飞书，粘贴到群里
  6. 每天重复，耗时约 15 分钟

使用 OpenClaw 的方式

  1. 一次性配置任务："每天早上 8 点，收集 AI 行业新闻，总结后发送到飞书群"
  2. OpenClaw 自动：
    • 定时唤醒（每天 8:00）
    • 打开指定新闻网站
    • 抓取最新文章标题和摘要
    • 调用大模型生成总结
    • 通过飞书 API 发送到群组
    • 记录执行日志
  3. 之后每天自动执行，无需人工干预
效率对比：
ChatGPT 方案：15 分钟/天 × 365 天 = 91 小时/年
OpenClaw 方案：30 分钟（一次性配置）+ 2 分钟/天（监控）× 365 天 = 13 小时/年
节省时间：85%！

C. 常见问题 FAQ

Q1: 安装失败怎么办？

A: 90% 的安装失败是网络问题。国内用户建议使用镜像源，或直接在云平台一键部署。

Q2: API 密钥安全吗？

A: OpenClaw 本地存储密钥，不会上传。但仍需妥善保管 config 文件，不要提交到 Git。

Q3: 如何选择 AI 模型？

A: 追求最强能力选 Claude Opus，追求性价比选 DeepSeek，国内部署选通义千问。

Q4: Skills 从哪里来？

A: ClawHub 市场有 13000+ 技能，也可自己开发或找人定制。

Q5: 会不会误操作删除文件？

A: 有可能！首次使用建议在测试环境，高危操作前做好备份。
